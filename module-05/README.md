### 3.# Module 05 – Modularization, Teamwork, and Git Collaboration

## Description
This module focuses on **collaborative development** and **code modularization**. The projects were developed in groups to simulate real-world teamwork, encouraging the use of GitHub for version control, task division, and code integration. Each participant was responsible for implementing specific functions and contributing to a shared codebase.

## Projects

### 1. Modularized Calculator
- **Learning focus:** Code modularization and team collaboration.
- **Project structure:**
  - Each team member implemented one or more functions
  - A central `index.js` file was created to aggregate all modules
- **Features:**
  - Basic arithmetic operations:
    - Addition
    - Subtraction
    - Multiplication
    - Division
  - Input validation using `parseFloat`
- **Key concepts practiced:**
  - `module.exports` and `require`
  - Separation of responsibilities
  - Code integration via GitHub
- Includes its own README with detailed explanations.

### 2. E-sports Project
- **Learning focus:** Applying modularization concepts to a different domain.
- **Concepts practiced:**
  - Team-based code organization
  - Modular structure similar to the calculator project
  - Shared responsibility across multiple files
- Follows the same collaborative and modular approach as the calculator.

### 3. Mission Manager
- **Learning focus:** Modular architecture and collaborative problem-solving.
- **Concepts practiced:**
  - Dividing logic into independent modules
  - Centralized index for module aggregation
  - Git-based teamwork and version control
- Implements the same modular principles introduced in this module.
