# Reverse string

**Module 2** exercise: reverse a string and count vowels.

- **reverse.js** – Uses `prompt-sync` for input; reverses with `split/reverse/join` and counts vowels.
- **reverseFor.js** – Reverses a string using a `for` loop (no dependencies).

Run: `npm install` then `node reverse.js` or `node reverseFor.js`.
