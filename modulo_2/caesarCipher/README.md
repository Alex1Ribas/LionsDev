# Caesar cipher

**Module 2** exercise: Caesar cipher and string utilities in plain JavaScript.

- **caesarCipher(text, shift)** – Encodes text by shifting letters in the alphabet.
- **removeDuplicates(text)** – Returns the string with duplicate characters removed.
- **countVowels(text)** – Counts total vowels and per-vowel (a, e, i, o, u).

Run: `node index.js`
