## Space Station:

1. Create a main directory called SpaceStation.
- mkdir SpaceStation

2. Inside SpaceStation, create modules for different sections of the space station: a) Habitation b) Laboratory c) Hangar
- mkdir SpaceStation/{Habitation,Laboratory,Hangar}

3. Inside Habitation, create numbered rooms from 1 to 3.
- mkdir SpaceStation/Habitation/{room1,room2,room3}

4. Inside the Laboratory, create areas called Research1 and Research2.
- mkdir SpaceStation/Laboratory/{Research1,Research2}

5. Inside Hangar, create spaces for three spaceships numbered 1 to 3.
- mkdir SpaceStation/Hangar/{Ship1,Ship2,Ship3}

6. Remove Hangar from the space station.
- cd SpaceStation
- rm -r Hangar

7. Inside SpaceStation, create a file called Welcome.txt with the following welcome message: "Welcome to the Space Station!".
- cd ..
- echo "Welcome to the Space Station" >> Welcome.txt

8. Make a copy of the Welcome.txt file to the Laboratory
- cp Welcome.txt SpaceStation/Laboratory

9. Change the content of the Welcome.txt file inside the Laboratory to "Welcome to the Space Station! This is the laboratory!" (BONUS)
- echo "Welcome to the Space Station! This is the laboratory!" > Welcome.txt
