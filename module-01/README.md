# Module 01 – Fundamentals and Linux Terminal Basics

## Description
This module focuses on the very first steps in programming and Linux terminal usage. It covers basic file manipulation commands, fundamental JavaScript concepts, and simple operations with variables. The goal is to build a solid foundation for further development in Node.js and backend programming.

## Projects / Exercises

### 1. `spaceStation.txt`
- **Learning focus:** Navigating the Linux terminal and manipulating files.
- **Commands practiced:**
  - `mkdir` – create directories
  - `cd` – change directories
  - `touch` – create files
  - `ls` – list directory contents
  - `pwd` – show current path
  - `echo` – write text to terminal
  - `rm` – remove files or directories

### 2. `fundamentals.js`
- **Learning focus:** Basic JavaScript data types and syntax.
- **Concepts covered:**
  - Data types: `string`, `number`, `boolean`
  - Variable declarations: `let`, `var`, `const`
  - Strings and template literals (using backticks `` ` `` and `$`)
  - Comparisons and negations
  - Using `console.log` for output

### 3. `processTwoNumbers.js`
- **Learning focus:** Working with variables and arithmetic operations in JavaScript.
- **Exercises included:**
  - Create two values using `let`
  - Perform addition, division, and multiplication
  - Store results in variables
  - Concatenate results into a final output
