## Description
This module deepens the understanding of **core JavaScript logic**, focusing on arrays, functions, loops, and objects. The exercises emphasize data manipulation, iteration, and problem-solving using different programming structures. This is a key step toward writing structured and maintainable backend code.

## Projects / Exercises

### 1. `01_crud.js`
- **Learning focus:** Basic CRUD operations using arrays.
- **Concepts practiced:**
  - Array manipulation with `push` - create
  - Using `console.log` to inspect data - read
  - Updating values using index assignment - update
  - Removing elements with `splice` - delete

### 2. `02_arrayMethod.js`
- **Learning focus:** Mastering built-in array methods.
- **Methods covered:**
  - `push`, `pop`
  - `shift`, `unshift`
  - `splice`, `slice`
  - `includes`, `indexOf`
  - `length`

### 3. `03_mytable.js`
- **Learning focus:** Practical array manipulation exercise.
- **Concepts practiced:**
  - Adding, removing, and deleting items
  - Applying array methods in a real scenario

### 4. `04_functions.js`
- **Learning focus:** Function structure and execution.
- **Concepts practiced:**
  - Function declaration and invocation
  - Parameters and return values
  - Arithmetic functions (sum of three numbers, division)
  - Arrow functions

### 5. `05_loop.js`
- **Learning focus:** Iteration and repetition logic.
- **Loops covered:**
  - `for`
  - `while`
  - `do...while`
  - `forEach`
- Includes explanatory comments for clarity.

### 6. `06_pontuation.js`
- **Learning focus:** Calculating averages from user input.
- **Concepts practiced:**
  - Arrays and numeric parsing with `parseFloat`
  - Reading input using `readline`
  - Using `push`, `length`, and index access
  - Average calculation logic

### 7. `07_objects.js`
- **Learning focus:** Object structure and property access.
- **Concepts practiced:**
  - Creating simple objects
  - Properties such as painting, author, name, year, and meaning
  - Accessing object properties using dot notation

### 8. `08_sumPairsImpairs.js`
- **Learning focus:** Conditional accumulation and numeric analysis.
- **Concepts practiced:**
  - Looping from 0 to 999
  - Modulo operator `%` to identify even and odd numbers
  - Separate accumulation and summation of even and odd values

### 9. `09_multiplicationTable.js`
- **Learning focus:** Loop-based arithmetic logic.
- **Concepts practiced:**
  - Generating a multiplication table using a `for` loop
